How to create a portfolio like mine

1. Login to GitHub.com.

2. Enter your GitHub username:

3. Go to https://github.com/new/import.

4. Repository url: https://github.com/Akleinikkink9587/Akleinikkink9587.github.io.git

5. Repository name: "YourUsername.github.io".

6. Click "Begin Import" and then wait for it to complete.

7. Edit _data/authors.yml.

8. Edit _includes/footer.html.

9. Edit _config.yml. (title, email, description, url)

10. Edit admin/config.yml. (collections -> feilds -> fields -> options)

11. Go to Netlify.com and log in with GitHub.

12. Click "New site from Git" and then select GitHub.

13. Select the "YourUsername.github.io" repository and then hit "Deploy Site".

14. Go to Settings -> General -> Site Details and change the site name to "YourUsername".

15. Go to Identity and click "Enable Identity".

16. Go to Settings -> Identity -> Registration and click "Edit settings".

17. Select "Invite only" and then click save.

18. Go to Settings -> Identity -> Services and click "Enable Git Gateway".

19. Go to Identity.

20. Click "Invite users", enter your email, click "Send" and then check your email for a site invitation.

21. Click "Accept the invite" in the email and then enter a password.

22. Go to the admin page (/admin).

23. Login with your email and the password you just created.

24. Make all of your desired changes using the CMS (Content Management System).

25. View your site: YourUsername.github.io.

Note: The CSS can be changed on GitHub at _sass/partials.
