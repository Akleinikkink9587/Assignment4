How to create a portfolio like mine

1. Login to GitHub.com.

2. Enter your GitHub username:

3. Go to https://github.com/new/import.

4. For the repository url enter: https://github.com/Akleinikkink9587/Akleinikkink9587.github.io.git

5. Name the repository "YourUsername.github.io".

6. Click "Begin Import" and then wait for it to complete.

7. Edit _data/authors.yml.

8. Edit _includes/footer.html.

9. Edit _config.yml.

10. Go to Netlify.com and log in with GitHub.

11. Click "New site from Git" and then select GitHub.

12. Select the "YourUsername.github.io" repository and then hit "Deploy Site".

13. Go to Settings -> General -> Site Details and change the site name to "YourUsername".

14. Go to Identity and click "Enable Identity".

15. Go to Settings -> Identity -> Registration and click "Edit settings".

16. Select "Invite only" and then click save.

17. Go to Settings -> Identity -> Services and click "Enable Git Gateway".

18. Go to Identity.

19. Click "Invite users" and then check your email for a site invitation.

20. Click "Accept the invite" in the email and then enter a password.

21. Go to the admin page (/admin).

22. Login with your email and the password you just created.

23. Make all of your desired changes using the CMS (Content Management System).

24. View your site at Netlify or GitHub pages (span class="italic userName">YourUsername.github.io).

Note: The CSS can be changed on GitHub at _sass/partials.
